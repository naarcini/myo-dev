boolean com.thalmic.myo.Hub.init ( Context  context,
String  applicationIdentifier 
) throws IllegalArgumentException

Initialize the Hub with an application identifier.

Must be called before any other usage. Will fail if the system doesn't support Bluetooth Low Energy.

Parameters
contexta Context from which an Application Context will be retrieved and held on to.
applicationIdentifiermust follow a reverse domain name format (ex. com.domainname.appname). Application identifiers can be formed from the set of alphanumeric ASCII characters (a-z, A-Z, 0-9). The hyphen (-) and underscore (_) characters are permitted if they are not adjacent to a period (.) character (i.e. not at the start or end of each segment), but are not permitted in the top-level domain. Application identifiers must have three or more segments. For example, if a company's domain is example.com and the application is named hello-world, one could use "com.example.hello-world" as a valid application identifier. Application identifiers must not be longer than 255 characters. applicationIdentifier can be an empty string.
Returns
true if initialization completed successfully, false otherwise.
Exceptions
IllegalArgumentExceptionif applicationIdentifier is not properly formatted.
